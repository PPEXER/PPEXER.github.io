เหตุผลที่อยากเรียน Full Stack Web Development

เนื่องจาก Full Stack Web Development เป็นทักษะที่มีความสำคัญในยุคดิจิทัล สามารถสร้างและจัดการเว็บไซต์ตั้งแต่ฝั่ง Front-end ไปจนถึง Back-end ได้ ทำให้มีโอกาสในการทำงานที่หลากหลาย อีกทั้งยังเพิ่มทักษะด้านการแก้ปัญหาและการคิดเชิงตรรกะที่มีประโยชน์ในการทำงานจริง

ความคาดหวังต่อวิชานี้

คาดหวังว่าจะได้รับความรู้ที่ครอบคลุมทุกด้านของการพัฒนาเว็บไซต์ สามารถพัฒนาแอปพลิเคชันที่สามารถทำงานได้จริง และได้มีโอกาสเรียนรู้เครื่องมือและเทคโนโลยีที่ทันสมัยในการพัฒนา

จุดอ่อนของตัวเองที่เกี่ยวกับทักษะ Programming และแผนการพัฒนาต่อไป

จุดอ่อนของตัวเองเกี่ยวกับการเขียนโปรแกรมคือความรู้และทักษะด้าน Back-end ที่ยังไม่แข็งแรงพอ ฉันจึงต้องการเน้นไปที่จุดอ่อนนี้ด้วยการศึกษาสถาปัตยกรรมโปรแกรมเพิ่มเติม รวมถึงการจัดการฐานข้อมูล การเรียกใช้ API และการพัฒนาเว็ปไซต์ที่พร้อมใช้งานในระดับจริง
